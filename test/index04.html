주요 내용

_includes 폴더의 공통적인 html을 추가하고 사용하는 방법

설명

1. 프로젝트 루트 폴더에 _includes폴더를 만들고 그 내부에 head4.html 파일 작성
2. head4.html에 내용 작성
3. 프로젝트 루트 폴더에 test폴더를 만들고 그 내부에 index04.html 파일 작성
4. index04.html에 추가하고 싶은 부분에 _include/head4.html 내용을 추가.

컴파일 후 생성 구조
