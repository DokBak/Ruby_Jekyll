주요 내용

CSS의 확장 기능인 sass을 사용하기

설명

1. 프로젝트 루트 폴더에 _sass폴더를 만들고 그 내부에 testing.scss 파일을 추가했다.
2. testing.scss에는 [.te { color: green; } .tt { color: green; }] []내용 추가
3. assets/css/cssStyles.scss 를 생성하고 머릿말 아래에 [ @import "testing" ]으로 testing css를 추가.
4. 테스트로 [.te { color: red; } ] []내용을 추가.
5. 기본적인 css는 루트 폴더에 있는 _sass의 내용이 적용
6. assets 폴더에 있는 scss에서 _sass의 scss를 호출(import)하여 사용하는데 import 밑에 css 설정 내용이 있다면 import 밑의 내용으로 덮어써진다.
7. 따라서 class가 te인 주요내용은 red, class가 tt인 설명은 green

컴파일 후 생성 구조

_site/assets/css/cssStyles.css
