_posts 폴더에 게시물들을 사용하는 방법

설명

1. 프로젝트 루트 폴더에 _layouts폴더를 만들고 그 내부에 layout5.html 파일 작성
2. layout5.html에 기본 폼으로 사용할 내용을 작성
3. {{ content }} 태그를 추가해서 기본 폼이외의 정보가 추가될 수 있도록한다.
3. 프로젝트 루트 폴더에 form.html 파일 작성
4. form.html의 상단 머릿말에 layout : layout5을 추가 5. layout을 제외하고도 layout5.html에서 필요로 되는 값을 추가해준다.
