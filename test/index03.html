주요 내용

_data 폴더의 data값을 사용하는 방법

설명

1. 프로젝트 루트 폴더에 test폴더를 만들고 그 내부에 index03.html 파일 작성
2. index03.html에 기본적인 html 내용 작성
3. 프로젝트 루트 폴더에 _data 폴더를 만들고 그 내부에 language.yml 파일을 작성
4. 데이터는 - 로 시작하고 - 가 없는 데이터는 같은 그룹의 데이터가 된다.
5. for문을 사용하여 language.yml 파일의 데이터를 취득한다.
6. for문 사용시 language.yml 파일값은 site.[폴더명].[yml파일명]으로 작성한다. 예) site.data.language

컴파일 후 생성 구조

_data/language.yml
_site/test/index03.html
