주요 내용

머릿말 추가 및 사용 방법 확인

설명

1. 프로젝트 루트 폴더에 test폴더를 만들고 그 내부에 index02.html 파일 작성
2. index02.html에 기본적인 html 내용 작성
3. html 최상단 <!doctype html> 위에 머릿말 구역을 만든다. --- 두줄을 추가한다.
4. 머릿말 구역에 title : index02을 작성한다. 이 때 title은 변수명, index02는 title의 값이 된다.
5. title 태그에 {{ page.title }}

컴파일 후 생성 구조

_site/test/index02.html

부호 출력

특수 문자 < > { }
코드 값 <(<) >(>) { }
